Frees memory previously allocated by Merkle_Malloc, with optional debug information.

This function frees the memory pointed to by d. If MERKLE_DEBUG is defined, it prints debug information about the deallocation.

Parameters
dPointer to the memory to free.
fileSource file name where the deallocation is requested.

void * Merkle_Malloc ( size_t size,
const char * file,
int line )

Allocates zero-initialized memory with optional debug information.

This function allocates a memory block of the given size, zero-initializes it, and, if MERKLE_DEBUG is defined, prints debug information about the allocation.

Parameters
sizeNumber of bytes to allocate.
fileSource file name where the allocation is requested.
lineLine number in the source file where the allocation is requested.
Returns
Pointer to the allocated memory, or NULL if allocation fails.
