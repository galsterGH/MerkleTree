Detailed Description

Public API for Merkle tree creation and destruction using SHA-256.

This header defines the error codes, opaque types, and functions for creating and destroying Merkle trees. The implementation uses SHA-256 and is suitable for cryptographic applications requiring data integrity verification.

Creates a Merkle tree from an array of data blocks.

Constructs a Merkle tree using SHA-256 hashing with the specified branching factor. The branching factor must be >= 2 and affects the tree's structure and height.

Parameters

sizesArray of sizes for each data block (must not be NULL).

Pointer to the created Merkle tree, or NULL on failure.

Pointer to the created Merkle tree, or NULL on failure.

Generates a Merkle proof for a leaf found using a custom finder function.

Searches the tree for a leaf matching the criteria defined by the finder function, then generates a proof path for the first matching leaf found. The search is performed in leaf index order (0 to leaf_count-1).

Parameters
treePointer to the Merkle tree (must not be NULL).
finderFunction pointer to locate the target leaf value (must not be NULL).
path_lengthPointer to store the length of the proof path (must not be NULL).
proofPointer to store the generated proof (must not be NULL).
Returns
MERKLE_SUCCESS on success, error code on failure.
Parameters
treePointer to the Merkle tree (must not be NULL).
finderFunction pointer to locate the target leaf value.
path_lengthPointer to store the length of the proof path.
Returns
Pointer to the generated proof on success, NULL on failure.

size_t leaf_index,

Generates a Merkle proof for a leaf at the specified index.

Creates a proof path from the specified leaf to the root. The proof can be used to verify that the leaf is part of the tree without reconstructing the entire tree. Returns NULL if the leaf index is out of bounds or if memory allocation fails.

Parameters
treePointer to the Merkle tree (must not be NULL).
leaf_indexIndex of the leaf to generate proof for (must be < leaf count).
proofPointer to store the generated proof (must not be NULL).
Returns
MERKLE_SUCCESS on success, error code on failure.
Parameters
treePointer to the Merkle tree (must not be NULL).
leaf_indexIndex of the leaf to generate proof for.
proofPointer to store the generated proof.
path_lengthPointer to store the length of the proof path.
Returns
MERKLE_SUCCESS on success, error code on failure.
